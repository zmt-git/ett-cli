## ett-cli

快速搭建nodejs的后台服务模板，模板采用express + ts + typeorm + mysql

### Global installation (全局安装)
```
npm install ett -g
```

### create project (创建项目模板)
```
ett create <project-name>
```

### Installation dependencies (安装依赖)
```
cd <project-name>
```
```
npm install or yarn
```

### Startup project (启动项目)
```
npm run dev or yarn dev
```
